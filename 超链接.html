基础超链接

在HTML中，超链接使用 <a> 标签创建，href属性指定链接的目标地址。

基本语法：

<a href="https://www.example.com">访问示例网站</a>

实际效果：

访问示例网站

不同类型的超链接

1. 外部链接

指向其他网站的链接，使用绝对URL：

<a href="https://www.google.com">访问Google</a>

2. 内部链接

指向自己网站内其他页面的链接，使用相对路径：

<a href="about.html">关于我们</a>

3. 锚点链接

指向同一页面特定位置的链接：

<a href="#section1">跳转到第一节</a>
...
<h2 id="section1">第一节</h2>

4. 邮件链接

点击会打开默认邮件客户端：

<a href="mailto:example@example.com">发送邮件</a>

5. 电话链接

在移动设备上点击会拨打电话：

<a href="tel:+1234567890">拨打电话</a>

6. 下载链接

添加download属性使链接成为下载链接：

<a href="document.pdf" download>下载PDF</a>
访问Google（外部链接） 跳转到基础部分（锚点链接） 发送邮件（邮件链接） 拨打电话（电话链接）

链接属性

target属性

控制链接的打开方式：

<a href="https://www.example.com" target="_blank">在新标签页打开</a>
<a href="https://www.example.com" target="_self">在当前页面打开（默认）</a>

rel属性

定义当前文档与链接文档的关系：

<a href="https://external.com" rel="noopener noreferrer">外部网站</a>
<a href="https://example.com" rel="nofollow">不要跟踪此链接</a>

title属性

提供链接的额外信息，鼠标悬停时显示：

<a href="https://example.com" title="访问示例网站">示例网站</a>
在新标签页中打开（带title提示）
安全提示： 使用target="_blank"时，建议同时添加rel="noopener noreferrer"以防止潜在的安全风险。

链接样式技巧

通过CSS，我们可以改变链接在不同状态下的外观：

a { color: #3498db; text-decoration: none; }
a:visited { color: #9b59b6; } /* 已访问的链接 */
a:hover { color: #2980b9; text-decoration: underline; } /* 鼠标悬停 */
a:active { color: #e74c3c; } /* 点击时的状态 */
